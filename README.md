# URL 압축 서비스기 <span style="font-size: 1.5rem">[링크](https://u-z.me) </span>

- Vue + vuetify 최소 사용으로 프론트 사이드 기능 구현 및 컴포넌트화
- Meteor(NodeJS) 프레임워크로 서버 구성

- DNS 적용 및 SSL(HTTPS) 활성화
- 구브라우저(IE) 크로스브라우징 대신 서버사이드 단계에서 별도 페이징 처리
